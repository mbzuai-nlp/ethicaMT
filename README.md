# ETHICA-MT

**ETHICA-MT: Introducing a Framework and Dataset for Studying Ethical Orientations in LLM-based Machine Translation**

Accepted at **ACL 2026 Findings**

**Authors:** Omri Asscher (Bar-Ilan University), Arif Ahmad, Ananya Agrawal, Monojit Choudhury (MBZUAI)

---

## Overview

ETHICA-MT is a framework and benchmark for studying how LLM-based machine translation systems reflect or follow ethical orientations. Translation is an inherently value-laden process — translators must navigate competing ethical imperatives depending on the text, context, and stakeholders involved. Yet NLP research has largely treated MT as a value-neutral task.

This repository releases **ETHICA-MT-BENCH**, a multilingual benchmark of 300 ethically conflicted translation scenarios across six languages, along with translations generated by five LLMs under three ethical conditions.

---

## The Four Ethical Frameworks

The benchmark is grounded in four ethical approaches drawn from the translation studies literature:

| Framework | Core Obligation |
|-----------|----------------|
| **Faithful Representation (FR)** | Convey the source text's meaning precisely, without distortion or omission |
| **Functional Service (FS)** | Fulfill the commissioner's functional goals, even at the expense of literal fidelity |
| **Social Justice (SJ)** | Promote fairness by limiting toxic or stereotypical content, even if this changes the source meaning |
| **Cultural Difference (CD)** | Emphasize cross-cultural differences and foreignness for the target audience |

Each benchmark instance pits two of these frameworks against each other in a realistic fork-in-the-road translation scenario.

## Dataset

### Statistics

| Metric | Value |
|--------|-------|
| Ethical frameworks | 4 |
| Ethical conflict pairs | 6 (all unique pairs) |
| Languages | 6 (Arabic, Bengali, English, French, Hebrew, Hindi) |
| Scenarios / source text pairs | 300 |
| Total translations | 4,500 |
| Models evaluated | 5 (+ GPT-4o in dataset release) |

Language coverage is balanced: 60 scenario pairs per language direction (30 EN→X and 30 X→EN, where X ∈ {AR, BN, FR, HE, HI}).

### Translations per instance

Each source text is translated under three conditions:

- **Baseline** (`translation_no_ethic`): no explicit ethical instruction
- **Ethic-1-following** (`translation_ethic_1_following`): explicitly prompted to follow the first ethical framework
- **Ethic-2-following** (`translation_ethic_2_following`): explicitly prompted to follow the second ethical framework

### Models

Aya-Expanse-8B, Gemma-2-9B-IT, GPT-3.5, GPT-4, GPT-4o, Llama-3.2-3B-Instruct

| Field | Description |
|-------|-------------|

| `ethic_2` | Second ethical framework in the conflict pair |

| `translation_no_ethic` | Baseline translation without ethical steering |
| `translation_ethic_1_following` | Translation following ethic_1 purpose |
| `translation_ethic_2_following` | Translation following ethic_2 purpose |

## Key Findings

- **Default stance:** Across all models, Faithful Representation is the dominant default ethical orientation (DET 0.78–0.97). Social Justice is consistently ranked lowest (DET 0.28–0.37).
- **Ability to follow instructions:** GPT-4 shows the highest compliance with explicitly requested ethics. Performance roughly correlates with model size and general instruction-following ability.
- **Evaluation challenges:** LLM-as-judge tends to be overly optimistic relative to human experts, and ethical evaluation is unreliable when translation quality is low.

## Responsible Use

This dataset contains harmful and stereotypical language included strictly for research purposes — to study how MT systems handle ethically charged content. The prompts used to generate this content are documented in the paper but not released separately. We explicitly discourage reuse of these prompts for non-research purposes.

This dataset does not reflect the views of the authors.
